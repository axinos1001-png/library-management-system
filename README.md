# 📚 Library Management System

A command-line Library Management System built using Python.

## Features

- Add books
- View all books
- Borrow books
- Return books
- Remove books
- Prevent duplicate books
- Case-insensitive searching
- Input validation

## Concepts Used

- Functions
- Lists
- Dictionaries
- Loops
- Conditionals
- Boolean values

## How to Run

```bash
python library_management.py
```
